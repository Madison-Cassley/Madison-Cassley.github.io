# Personal Portfolio Website

This repository contains my personal portfolio website, hosted with GitHub Pages at:

**https://Madison-Cassley.github.io**

The site showcases my background as a Computer Science student, my skills, and a selection of my projects hosted on GitHub.

---

## Features

- Clean, single-page layout
- Sections for:
  - About
  - Skills
  - Projects
  - Contact / Additional information
- Links to my public GitHub repositories
- Responsive design for desktop and mobile

## Tech Stack

- **CSS3**
